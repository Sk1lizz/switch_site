### Инструкция к использованию:

- Перечень команд:
  - pip install requests, bs4, pyfiglet
  - git clone https://github.com/Sk1lizz/switch_site.git
  - cd switch_site
  - python main.py

- 2 -> 1 - активировать программу; 2 - остановить работу программы; 3 - добавить сайт; 4 - удалить сайт; 0 -> 0 - выход и выклчение программы
- Иногда блокировка начинается не сразу, а через пару (1-5) минут.
- Если программа активирована, то при добавление сайта он не будет блокироваться. (Ошибка будет исправлена в следубющих патчах)
